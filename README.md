# cursor-clone

link:- https://cursor-clone-rohit667.netlify.app/
A frontend clone of the Cursor AI website, built using pure HTML and CSS.
This project focuses on recreating the modern UI, layout, and visual style of the Cursor homepage.

DISCLAIMER:
This project is created strictly for educational and practice purposes.
All trademarks, logos, images, and design inspiration belong to Cursor.
This project is not affiliated with or endorsed by Cursor.

TECH STACK:
- HTML5
- CSS3
- Google Fonts
- No JavaScript
- No frameworks or libraries

PROJECT STRUCTURE:
cursor-clone/
├── index.html
├── style.css
├── Images/
└── README.md

FEATURES:
- Modern dark-themed UI
- Clean navigation bar
- Hero section with call-to-action
- Feature sections inspired by Cursor
- Responsive layout (desktop-first)
- Structured and readable CSS

HOW TO RUN LOCALLY:
1. Clone the repository from GitHub
2. Open the project folder in VS Code
3. Open index.html in any modern browser

LEARNING OUTCOMES:
- Improved frontend layout skills
- Practice cloning real-world SaaS websites
- Better understanding of modern UI patterns
- Hands-on experience with HTML & CSS
